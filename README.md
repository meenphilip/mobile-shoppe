# Mobile Shoppe

A mobile-first ecommerce website template. Includes HTML, CSS, SASS, and JavaScript for a responsive product listing and cart UI.

## Features
- Responsive mobile-first layout
- Product listing, product detail and cart pages
- Sass source files (sass/) and compiled CSS (style.css)

## Prerequisites
- Node.js (for npm scripts)
- Dart Sass (either installed globally or as an npm dev dependency)

## Install & run
1. Install dev dependency for Sass (optional if installed globally):
   npm install --save-dev sass

2. Watch Sass and compile to CSS:
   npm start

3. Open index.html in a browser or serve the folder with a static server (e.g., npx serve .)
